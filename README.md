rc
==

My `rc`[^rc] files


## Setup

Clone

```sh
git clone git@github.com:yyolk/rc --recursive
```

Run the install script (uses [`dotbot`][dotbot]).

```sh
./install
```


[^rc]: `rc` means **R**un **C**ommands - https://en.wikipedia.org/wiki/RUNCOM


[dotbot]: https://github.com/anishathalye/dotbot
